# TypeScript

## 是什么----规范声明变量，约束语法,提示

1.一个 JavaScript 的超集

2.TypeScript 拓展了 JavaScript，并添加了类型

3.以 javascript 为基础构建的语言

4.跨域在任何支持 JavaScript 的普通中执行

5.TS 不能被 JS 解析器直接执行

## 增加内容

1.类型

2.支持 ES 的新特性

3.丰富的配置选项

4.添加 ES 不具的新特性

## 安装

1.node 环境

2.npm i -g typescript

3.tsc 进行编译

4.执行编译 tsc index.ts

5.自动编译 tsc -w
